# DB設計

- アプリはDBの表現形

## RDBMSの種類

- MySQL
    - オープンソース、商標個人問わず無料
    - 基本的な機能は全てある
- MariaDB
    - MySQLの上位互換
    - 大規模向けの機能が追加されている
- Postgresql
    - ビジネス向け
    - テーブルのネストとかもできる
- OracleDB
    - 大規模向け
- AWS Aurora
    - とにかく処理速度が早い
    - 互換性が高い
    - Durability◎

## 論理削除・物理削除

- 物理削除
    - 一度削除すると戻せない
- 論理削除
    - 削除した体にする(それ用のカラムを作り、値で判断)
